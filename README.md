# TIC-TAC-TOE-JISAAC007

A simple console-based Tic Tac Toe game written in C++.

## About This Project

This project was fully coded by me as part of my C++ learning journey.
I built it from scratch to practice fundamental programming concepts such as loops, conditionals, arrays, and functions.

The project is currently a work in progress, with core gameplay implemented and additional features still being developed.

## Current Features

* 3x3 board rendering in the console
* Two-player turn-based system
* Player input using coordinates (e.g., A1, B2)
* Dynamic board updates after each move
* Move validation(prevent overwriting cells)

## Project Status

🚧 Work in Progress

Planned features:

* Win condition detection
* Draw detection

## What I Learned

* Representing a game board using 2D arrays
* Mapping user input to array indices
* Structuring programs using functions
* Managing turn-based logic
